# ad-exposure

[![NPM version](https://img.shields.io/npm/v/ad-exposure?color=a1b858&label=)](https://www.npmjs.com/package/ad-exposure)

## install

```bash
npm install ad-exposure
```

## usage

```js

adExposure({
  ele: dom,
  callback: (entry) => {},
})
```
